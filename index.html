CibusSanus

Программный модуль учёта продаж ресторана европейской кухни

Описание

CibusSanus — приложение для учёта продаж ресторана. Работает автономно, база данных SQLite встроена и создаётся автоматически при первом запуске. Установка PostgreSQL не требуется.

Возможности

  • Управление категориями блюд (добавление, редактирование, удаление, поиск)
  • Управление блюдами (CRUD, фильтрация по активности)
  • Создание, редактирование и удаление заказов
  • Автоматический расчёт стоимости заказа
  • Сортировка списков по разным параметрам
  • Экспорт данных в Excel (.xlsx)
  • Формирование отчётов (выручка, чеки, загруженность, ТОП блюд)
