# CibusSanus

Программный модуль учёта продаж ресторана европейской кухни.

---

## Описание

CibusSanus — автономное приложение для автоматизации учёта продаж ресторана. Работает без внешних серверов: встроенная база данных SQLite создаётся автоматически при первом запуске.

### Возможности

- Управление категориями блюд (CRUD, поиск)
- Управление блюдами (CRUD, поиск, фильтрация по активности)
- Создание, редактирование и удаление заказов
- Автоматический расчёт стоимости заказа
- Сортировка по всем параметрам (имя, цена, дата, сумма и др.)
- Экспорт данных в Excel (.xlsx)
- Формирование отчётов с выгрузкой в Excel
- Статистика: выручка, средний/максимальный/минимальный чек
- Загруженность по часам, ТОП-10 блюд
- Полностью автономная работа (не нужен интернет/сервер)

## Установка и запуск

### Способ 1: Скачать готовый exe/apk

1. Перейдите в раздел **Releases** репозитория [cibussanus](https://github.com/cherepenkintimur-ctrl/cibussanus/releases)
2. Скачайте нужный файл:
   - `windows-runner.exe` — для Windows
   - `app-release.apk` — для Android
3. Запустите. База данных создастся автоматически при первом запуске.

### Способ 2: Запустить из исходников

```bash
# Клонировать репозиторий
git clone https://github.com/cherepenkintimur-ctrl/cibussanus.git
cd cibussanus

# Установить зависимости
flutter pub get

# Запустить
flutter run
```

Требуется установленный [Flutter SDK](https://docs.flutter.dev/get-started/install).

## Как это работает

При первом запуске приложение автоматически:

1. Создаёт файл `restaurant.db` в локальной папке приложения
2. Создаёт таблицы: `categories`, `dishes`, `orders`, `order_items`
3. Заполняет seed-данными: 7 категорий, 75 блюд, 30 заказов, 121 позиция заказа

Все данные хранятся локально. Подключение к внешним серверам не требуется.

## Экспорт в Excel

На экранах **Категории**, **Блюда**, **Заказы** и **Отчёт** доступна кнопкаэкспорта в Excel (.xlsx).

Экспорт отчёта включает 3 листа:
- **Сводка** — выручка, средний/максимальный/минимальный чек
- **По часам** — нагрузка по 24 часам
- **ТОП блюд** — рейтинг блюд по количеству продаж

---

## Сортировка

На экранах списков доступны чипсы-фильтры для сортировки:

| Экран | Параметры сортировки |
|---|---|
| Категории | По имени, по ID, по дате |
| Блюда | По имени, по цене, по категории, по ID |
| Заказы | По дате, по сумме, по номеру, по способу оплаты |

Нажатие на уже активный чип меняет направление сортировки (↑/↓).

## База данных

Файл `database/restaurant_db_dump.sql` содержит полную структуру PostgreSQL-схемы (архивная ссылка для справки). В текущей версии приложение использует SQLite и не зависит от PostgreSQL.

### Таблицы

| Таблица | Описание |
|---|---|
| `categories` | Категории блюд (7 записей) |
| `dishes` | Блюда (75 записей) |
| `orders` | Заказы (30 записей) |
| `order_items` | Позиции заказов (121 запись) |
